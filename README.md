# ToDownLoad WordPress插件

## 插件简介
ToDownLoad是一个简单易用的WordPress下载管理插件，帮助网站管理员更好地管理和展示下载资源。该插件由老蒋和他的小伙伴开发维护。

## 主要功能
- 文章内嵌下载框
- 自定义下载按钮样式
- 支持文件大小显示
- 支持版本号显示
- 支持解压密码提示
- 可自定义下载框颜色和样式
- 支持短代码调用

## 系统要求
- WordPress 5.0 或更高版本
- PHP 7.0 或更高版本
- 支持短代码的WordPress主题

## 安装方法
1. 下载插件压缩包
2. 在WordPress后台进入"插件 > 安装插件"
3. 点击"上传插件"按钮
4. 选择下载的压缩包文件
5. 点击"现在安装"
6. 安装完成后点击"启用插件"

## 使用说明
### 基本设置
1. 在WordPress后台进入"设置 > ToDownLoad"
2. 配置基本设置：
   - 启用/禁用插件
   - 设置解压密码提示文本
   - 自定义下载框样式

### 在文章中使用
1. 编辑文章时，在文章编辑器下方找到"下载设置"面板
2. 启用下载功能
3. 填写下载信息：
   - 下载标题
   - 按钮文字
   - 下载链接
   - 文件大小
   - 版本号

### 使用短代码
在文章中使用以下短代码：
```
[todownload]
```

## 自定义样式
插件支持自定义以下样式：
- 边框颜色
- 背景颜色
- 按钮颜色
- 标题颜色
- 标题大小
- 按钮文字大小

## 注意事项
- 请确保服务器有足够的存储空间
- 建议定期备份下载文件
- 请遵守相关法律法规，不要上传侵权内容
